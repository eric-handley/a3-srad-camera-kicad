# IMX290 Image Sensor

## Overview
Sony IMX290LQR-C CMOS Image Sensor
- 1/2.8-inch diagonal (Type 1/2.8)
- 2.13M effective pixels (1945H × 1097V)
- MIPI CSI-2 interface (2-lane or 4-lane)
- Parallel CMOS interface
- LVDS interface
- 10-bit / 12-bit ADC

## Power Requirements

### AVDD Rail (Analog Supply)
- **Voltage:** 2.9V (typical), range 2.80V - 3.00V
- **Standby Current:** 0.1 mA (max)
- **Record Current (CSI-2, 1080p60, 12-bit):**
  - Typical: 55 mA
  - Maximum: 111 mA

### OVDD Rail (Interface I/O Supply)
- **Voltage:** 1.8V (typical), range 1.70V - 1.90V
- **Standby Current:** 0.1 mA (max)
- **Record Current (CSI-2, 1080p60, 12-bit):**
  - Typical: 1 mA
  - Maximum: 2 mA

### DVDD Rail (Digital Supply)
- **Voltage:** 1.2V (typical), range 1.10V - 1.30V
- **Standby Current:** 14.0 mA (max)
- **Record Current (CSI-2, 1080p60, 12-bit):**
  - Typical: 94 mA
  - Maximum: 143 mA

## Total Power Consumption

### Standby Mode
- **Total Current:** 14.2 mA (max)
- **Total Power:** ~17 mW

### Record Mode (1080p60 H.265 Encoding)
- **Typical Total Current:** 150 mA equivalent @ 2.9V
- **Maximum Total Current:** 256 mA equivalent @ 2.9V
- **Typical Total Power:** ~200 mW
- **Maximum Total Power:** ~370 mW

## Notes
- Current values are for CSI-2 4-lane mode; 2-lane implementation may have slightly lower consumption
- Maximum values measured at worst-case conditions: 3.00V/1.9V/1.3V supplies, Tj=60°C
- Standby measured with INCK=0V, light-obstructed state
- Interface includes I2C control bus (powered from 1.8V OVDD rail)
